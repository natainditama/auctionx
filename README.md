## ğŸš€ Getting Started

<!-- Prerequisites -->
### ğŸ”§ Prerequisites

- [Composer](https://getcomposer.org/)
- [MySQL](https://www.mysql.com/)

<!-- Run Locally -->
### ğŸƒ Run Locally

Clone the project

```bash
  git clone https://github.com/natainditama/auctionx.git
```

Go to the project directory

```bash
  cd auctionx
```

Import the database schema

```bash
  mysql -u your_username -p < path/to/auctionx.sql
```
Replace your_username and path/to/auctionx.sql with your actual MySQL username and the path to the auctionx.sql file

Generate the autoload file

```bash
  composer dump-autoload
```

Start the php server

```bash
  php -S localhost:5000
```

Open http://localhost:5000/public in your web browser
